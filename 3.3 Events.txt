const title = document.querySelector(".hello h1");

function handleTitleClick () {
    if (title.style.color === "") {
        title.style.color = "blue";
    } else {
        title.style.color = "";
    }
    console.log("Title was clicked.");
}

title.addEventListener("click", handleTitleClick);
===================================
내가 가져온 요소에 event를 추가하고 싶다면 addEventListener 함수를 사용하면 된다.
addEventListener 함수의 인자로는 [첫 번째 인자는 어떠한 이벤트가 발생했을 때] 이고,
 [두 번째 인자는 해당 이벤트가 발생했을 때 실행할 함수를 넣는다.] .

※ event를 찾는 방법 ※
1. MDN의 HTMLHeadingElement
2. console.dir 을 이용하여 on 으로 시작되는 것들 살펴보기

※ event 종류 ※
(HTML)
1. click
2. mouseenter
3. mouseleave
(Window)
4. resize
5. copy
6. online
7. offline
...

addEventListener 대신에 이렇게 사용할 수도 있다. title.onClick = handleTItleClick;
즉, "title.addEventListener("click", handleTitleClick);" 와 "title.onClick = handleTItleClick;" 는
같은 기능을 수행한다. 이처럼 같은 기능을 수행함에도 불구하고 addEventListener를 사용하는
이유는 removeEventListener로 해당 이벤트를 제거할 수 있기 때문이다.
